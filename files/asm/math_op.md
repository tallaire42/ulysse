# Math Operation List

## Math Operation List signed

- add a, b	a	= a + b
- sub a, b	a	= a - b
- mul reg	rax	= rax * reg
- div reg	rax	= rax / reg
- neg neg	reg	= -reg
- inc reg	reg	= reg + 1
- dec reg	reg	= reg - 1
- adc a, b	a 	= a + b + CF
- sbb a, b	a	= a - b - CF

## Math Operation List unsigned

- imul reg	rax	= rax * reg
- div reg	rax	= rax / reg
